# Applied Research Methods Website

This repository hosts the website for the 2026 fall course **Applied Research Methods**. The site is designed to be accessible, responsive, and easy to update.

**Live Site:** [https://the-oak-lab.github.io/arm-fall2026](https://the-oak-lab.github.io/arm-fall2026)

### Running Locally

If you want to test changes before pushing:

1. **Clone the Repo:**
   ```bash
   git clone https://github.com/the-oak-lab/arm-fall2026.git
   ```

2. **Serve Locally:**
   You can use a simple HTTP server. For example, with Python:
   ```bash
   python3 -m http.server 8000
   ```
   Then open `http://localhost:8000` in your browser.
